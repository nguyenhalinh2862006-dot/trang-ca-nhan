# Ý tưởng thiết kế tổng thể

Ý tưởng thiết kế website hướng đến:
- Phong cách hiện đại.
- Tối giản nhưng sáng tạo.
- Giao diện dễ nhìn và dễ sử dụng.
- Tập trung vào trải nghiệm người dùng.

Website sử dụng:
- Tông màu tối kết hợp màu nhấn nổi bật.
- Font chữ hiện đại.
- Hiệu ứng chuyển động nhẹ.
- Bố cục rõ ràng và trực quan.
Mục tiêu là tạo ra một portfolio cá nhân mang dấu ấn riêng và phù hợp với lĩnh vực Đồ họa Kỹ thuật số.

- Tối ưu giao diện responsive nâng cao
- Thêm animation chuyên nghiệp
- Tích hợp form liên hệ
- Hoàn thiện portfolio chuyên nghiệp hơn
